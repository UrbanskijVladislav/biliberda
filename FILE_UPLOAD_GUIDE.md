# 📎 Система завантаження файлів

## ✨ Можливості

Тепер ти можеш надсилати файли AI асистенту! Система підтримує:

- 🖼️ **Зображення**: JPG, PNG, GIF, WebP
- 🎥 **Відео**: MP4, AVI, MOV, WebM
- 📄 **Документи**: PDF, DOC, DOCX, TXT
- 📦 **Архіви**: ZIP, RAR, 7Z
- 🎵 **Аудіо**: MP3, WAV, OGG

## 🚀 Як використовувати

### 1. Прикріпити файл

Натисни кнопку 📎 біля поля вводу або перетягни файл в поле вводу.

### 2. Переглянути прикріплені файли

Після вибору файлу з'явиться превью з:
- Іконкою типу файлу
- Назвою файлу
- Розміром файлу
- Кнопкою видалення (×)

### 3. Відправити

Натисни кнопку відправки або Enter. Файли будуть завантажені разом з повідомленням.

## ⚙️ Технічні деталі

### Обмеження

- **Максимальний розмір файлу**: 10 MB
- **Кількість файлів**: Необмежена (але розумно використовуй)
- **Типи файлів**: Всі основні типи підтримуються

### Безпека

- Всі імена файлів очищаються від небезпечних символів
- Файли зберігаються в окремій папці `uploads/`
- Папка `uploads/` додана до `.gitignore`

## 🔧 Backend API

### POST /chat (з файлами)

**Request:**
```
Content-Type: multipart/form-data

message: "Текст повідомлення"
language: "uk" або "en"
file_0: [файл 1]
file_1: [файл 2]
...
```

**Response:**
```json
{
  "response": "Відповідь AI",
  "personality": "normal",
  "emoji": "🤖",
  "name": "Звичайний",
  "files_received": 2
}
```

### GET /uploads

Повертає список завантажених файлів:

```json
{
  "files": [
    {
      "name": "example.jpg",
      "size": 1024000,
      "modified": 1234567890.123
    }
  ]
}
```

### GET /uploads/<filename>

Повертає файл для завантаження.

### DELETE /uploads/<filename>

Видаляє файл з сервера.

## 📋 Frontend методи

### handleFileSelect(event)
Обробляє вибір файлів користувачем.

### renderAttachedFiles()
Відображає превью прикріплених файлів.

### removeFile(index)
Видаляє файл зі списку прикріплених.

### sendMessageWithFiles()
Відправляє повідомлення з файлами на сервер.

### addFilesMessage(files)
Додає повідомлення з файлами в чат.

### getFileIcon(type, name)
Повертає емодзі іконку для типу файлу.

### formatFileSize(bytes)
Форматує розмір файлу (B, KB, MB, GB).
