# 🎨 Стильне модальне вікно

## ✅ Що змінено

Замінено стандартні браузерні `alert()` та `confirm()` на красиві модальні вікна в стилі сайту!

## 🎯 Типи модальних вікон

### 1. Інформація (Info) ℹ️
```javascript
ui.showInfo('Це інформаційне повідомлення');
```

### 2. Успіх (Success) ✅
```javascript
ui.showSuccess('Операція виконана успішно!');
```

### 3. Помилка (Error) ❌
```javascript
ui.showError('Виникла помилка при виконанні');
```

### 4. Попередження (Warning) ⚠️
```javascript
ui.showWarning('Будьте обережні!');
```

### 5. Підтвердження (Confirm) ❓
```javascript
ui.showConfirm('Ви впевнені?', () => {
    console.log('Підтверджено!');
});
```

## 📝 Приклади використання

### Очищення чату
```javascript
clearChat() {
    this.showConfirm(
        'Ви впевнені, що хочете очистити чат?',
        () => {
            // Очищення чату
            this.chatMessages.innerHTML = '';
            this.showSuccess('Чат очищено!');
        },
        'Очистити чат?'
    );
}
```

### Очищення пам'яті
```javascript
async clearMemory() {
    this.showConfirm(
        'Всі збережені дані будуть видалені.',
        async () => {
            try {
                await fetch('/memory', {method: 'DELETE'});
                this.showSuccess('Пам\'ять очищено!');
            } catch (error) {
                this.showError('Помилка очищення');
            }
        },
        'Очистити пам\'ять?'
    );
}
```

## 🚀 Як використовувати

### Базове використання
```javascript
// Просте повідомлення
ui.showInfo('Привіт!');

// З власним заголовком
ui.showSuccess('Дані збережено', 'Успіх!');

// Підтвердження з callback
ui.showConfirm('Видалити файл?', () => {
    deleteFile();
});
```

### Розширене використання
```javascript
ui.showModal({
    type: 'warning',
    title: 'Увага!',
    message: 'Це важлива дія',
    icon: '⚠️',
    confirmText: 'Продовжити',
    cancelText: 'Відміна',
    showCancel: true,
    onConfirm: () => console.log('OK'),
    onCancel: () => console.log('Cancel')
});
```

## 🎉 Результат

Тепер замість стандартних браузерних вікон:
- ❌ `alert('Повідомлення')`
- ❌ `confirm('Підтвердити?')`

Використовуються стильні модальні вікна:
- ✅ `ui.showInfo('Повідомлення')`
- ✅ `ui.showConfirm('Підтвердити?', callback)`
