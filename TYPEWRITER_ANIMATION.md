# ⌨️ АНІМАЦІЯ ДРУКАРСЬКОЇ МАШИНКИ

## ✨ ЩО ДОДАНО:

### 1. Ефект друкування для AI відповідей
- ✅ Текст з'являється посимвольно (як друкарська машинка)
- ✅ Миготливий курсор під час друкування
- ✅ Автоматичне прокручування вниз
- ✅ Швидше друкування для пробілів

### 2. Візуальні ефекти
- ✅ Анімований курсор `|` що мигає
- ✅ Плавне зникнення курсора після завершення
- ✅ Збереження форматування тексту (жирний, курсив)

### 3. Налаштування швидкості
- Базова швидкість: 30ms на символ
- Пробіли: 15ms (швидше)
- Можна змінити в коді: `speed` параметр

## 🎬 ЯК ЦЕ ПРАЦЮЄ:

### Для користувача:
- Повідомлення з'являються миттєво (без анімації)

### Для AI:
1. Створюється порожнє повідомлення з курсором
2. Текст додається посимвольно
3. Курсор мигає під час друкування
4. Після завершення курсор зникає

## 🔧 НАЛАШТУВАННЯ:

### Змінити швидкість друкування:
В `script.js` знайди метод `typeWriter` і зміни параметр `speed`:

```javascript
typeWriter(text, element, cursor, speed = 30) // 30ms = швидко
typeWriter(text, element, cursor, speed = 50) // 50ms = повільніше
typeWriter(text, element, cursor, speed = 10) // 10ms = дуже швидко
```

### Вимкнути анімацію:
Замість `this.typeWriter(...)` використай:
```javascript
element.innerHTML = this.formatMessage(content);
if (cursor) cursor.remove();
```
