# 🎙️ Налаштування ElevenLabs AI TTS

## Чому ElevenLabs?

✅ **Природний голос** - звучить як жива людина, не робот
✅ **Безкоштовно** - 10,000 символів на місяць (достатньо для тестування)
✅ **Українська мова** - підтримує українську та англійську
✅ **Швидко** - генерація за 1-2 секунди

## Як отримати безкоштовний API ключ?

### Крок 1: Реєстрація

1. Відкрий https://elevenlabs.io/
2. Натисни **"Sign Up"** (Зареєструватися)
3. Введи email та пароль (або увійди через Google)
4. Підтверди email

### Крок 2: Отримання API ключа

1. Після входу натисни на свій профіль (правий верхній кут)
2. Вибери **"Profile"** → **"API Keys"**
3. Натисни **"Create API Key"**
4. Скопіюй ключ (виглядає як `sk_...`)

### Крок 3: Додавання ключа в проект

1. Відкрий файл `.env`
2. Знайди рядок:
   ```
   ELEVENLABS_API_KEY=your_elevenlabs_api_key
   ```
3. Заміни на свій ключ:
   ```
   ELEVENLABS_API_KEY=sk_твій_ключ_тут
   ```
4. Увімкни AI TTS:
   ```
   USE_AI_TTS=True
   ```
5. Збережи файл
6. Перезапусти сервер (`python run.py`)

## Перевірка роботи

1. Відкрий http://localhost:5000
2. Натисни 🔇 щоб увімкнути озвучку
3. Напиши "Привіт"
4. Асистент відповість **природним AI голосом** замість роботизованого!

## Безкоштовний план

- **10,000 символів/місяць** - це приблизно:
  - 200-300 відповідей асистента
  - Достатньо для тестування та демо
  
- Якщо закінчаться символи:
  - Автоматично перемкнеться на браузерний TTS
  - Або можна оновити план ($5/місяць за 30,000 символів)

## Альтернативи (якщо не хочеш реєструватися)

Якщо не хочеш використовувати ElevenLabs, залиш:
```
USE_AI_TTS=False
```

Тоді буде використовуватися звичайний браузерний TTS (безкоштовно, але роботизований).

## Технічні деталі

- **Модель**: eleven_multilingual_v2
- **Голоси**: 
  - Українська: Adam (чоловічий)
  - Англійська: Rachel (жіночий)
- **Формат**: MP3
- **Передача**: Base64 (без збереження файлів)
- **Fallback**: Автоматично перемикається на браузерний TTS при помилці
