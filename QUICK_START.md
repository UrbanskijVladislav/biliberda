# 🚀 Швидкий старт - Нові функції

## ⚡ Швидкі команди

### Як почати?
1. Відкрийте http://localhost:5000
2. Введіть `/` в поле вводу
3. Побачите список команд з автодоповненням!

### Топ-5 команд для початку:

```
/help      → Показати всі команди
/weather   → Погода в вашому місті
/joke      → Розказати жарт
/calc 2+2  → Калькулятор
/stats     → Ваша статистика
```

## 💡 Автодоповнення

- Почніть вводити `/` - з'явиться список команд
- Використовуйте ↑↓ для навігації
- Натисніть `Tab` для вибору
- Натисніть `Enter` для відправки

## 🌐 API функції

### Погода
```
/weather Київ
/weather London
/weather Paris
```

### Переклад
```
/translate Hello world
/translate Привіт світ
```

### Курси валют
```
/currency USD
/currency EUR
```

### Калькулятор
```
/calc 15*20
/calc (100+50)/2
/calc 3.14*2
```

## 🎭 Особистості

Змініть особистість асистента:
```
/mood
```

Кожна особистість відповідає по-своєму!

## 🌍 Мови

Перемкнути мову:
```
/lang
```

Або натисніть кнопку 🇺🇦/🇬🇧

## 📊 Статистика

Подивіться свою статистику:
```
/stats
```

## 🧹 Очистити чат

```
/clear
```

## 🎉 Спробуйте зараз!

1. Введіть `/weather` - дізнайтеся погоду
2. Введіть `/joke` - посміхніться
3. Введіть `/calc 2+2` - порахуйте
4. Введіть `/mood` - змініть особистість
5. Введіть `/help` - побачите всі команди
