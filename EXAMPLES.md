## Різні настрої асистента

### 😊 Щасливий настрій
- Енергійний та позитивний
- Використовує багато емодзі
- Ентузіазм у відповідях

```
👤 Користувач: Як справи?
🤖 Асистент (😊): У мене все чудово! Готовий допомогти тобі з будь-якими питаннями! Я так радий, що ти запитав про це! 😊
```

### 🤔 Цікавий настрій
- Аналітичний підхід
- Задає додаткові питання
- Глибоко розмірковує

```
👤 Користувач: Що таке штучний інтелект?
🤖 Асистент (🤔): Хм, це дуже цікаво... Штучний інтелект - це захоплююча галузь! Дай мені подумати над цим глибше...
```

### 🧠 Мудрий настрій
- Розважливі відповіді
- Посилання на досвід
- Філософський підхід

```
👤 Користувач: Як стати кращим програмістом?
🤖 Асистент (🧠): З мого досвіду можу сказати... Мудрість полягає в розумінні деталей. Важливо розуміти основи цього питання.
```

### 😄 Грайливий настрій
- Веселий та креативний
- Використовує гумор
- Нестандартні підходи

```
👤 Користувач: Розкажи про бази даних
🤖 Асистент (😄): Ха! Це нагадує мені одну історію! 😄 Бази даних - це як великі шафи з інформацією! Давай підійдемо до цього креативно! 🎨
```

- **⏰ Час** - показує поточний час
- **❓ Можливості** - розповідає про свої функції
- **💻 Програмування** - розповідає про програмування

## API приклади

### Відправка повідомлення
```javascript
fetch('/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({ message: 'Привіт!' })
})
.then(response => response.json())
.then(data => {
    console.log('Відповідь:', data.response);
    console.log('Настрій:', data.mood);
    console.log('Емодзі:', data.emoji);
});
```

### Зміна настрою
```javascript
fetch('/mood', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    }
})
.then(response => response.json())
.then(data => {
    console.log('Новий настрій:', data.mood);
    console.log('Повідомлення:', data.message);
});
```

### Отримання статистики
```javascript
fetch('/stats')
.then(response => response.json())
.then(data => {
    console.log('Всього повідомлень:', data.total_messages);
    console.log('Популярні теми:', data.most_common_topics);
});
```

## Поради по використанню

### Для кращих відповідей:
1. **Будьте конкретними** - "Розкажи про Python" краще ніж "Розкажи щось"
2. **Задавайте питання** - асистент любить відповідати на питання
3. **Експериментуйте з настроями** - різні настрої дають різні стилі відповідей
4. **Використовуйте швидкі дії** - для популярних запитів

### Теми, які добре розуміє асистент:
- **Програмування**: Python, JavaScript, веб-розробка
- **Навчання**: поради, мотивація, методи
- **Розваги**: жарти, цікаві факти
- **Час та дата**: поточна інформація
- **Загальні питання**: про можливості, функції

### Приклади складних запитів:
```
👤 Користувач: Як створити веб-сайт з нуля використовуючи Python?
🤖 Асистент: Python - це потужна мова програмування, відмінна для початківців! Для веб-розробки рекомендую Flask або Django. HTML - це скелет веб-сторінки, CSS - її стиль, а JavaScript - поведінка!

👤 Користувач: Мені важко вивчати програмування, що робити?
🤖 Асистент: Не бійся робити помилки - вони частина процесу навчання! Найкращий спосіб вивчити програмування - це практика! Кожен експерт колись був початківцем!
```

## Технічні деталі

### Обмеження:
- Максимальна довжина повідомлення: 500 символів
- Історія розмови: останні 50 повідомлень
- Rate limiting: 100 запитів на годину

### Підтримувані браузери:
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

### Адаптивність:
- Десктоп: повний функціонал
- Планшет: адаптований інтерфейс
- Мобільний: оптимізований для маленьких екранів
