# 📎 Швидкий старт: Завантаження файлів

## 🎯 Що нового?

Тепер можеш надсилати файли AI асистенту! Система готова до використання.

## 🚀 Як користуватися

### Крок 1: Відкрий асистента
```
http://localhost:5000
```

### Крок 2: Прикріпи файл
- Натисни кнопку 📎 біля поля вводу
- Або перетягни файл в поле вводу

### Крок 3: Відправ
- Напиши повідомлення (опціонально)
- Натисни Enter або кнопку відправки

## 📋 Підтримувані типи файлів

| Тип | Формати | Іконка |
|-----|---------|--------|
| Зображення | JPG, PNG, GIF, WebP | 🖼️ |
| Відео | MP4, AVI, MOV, WebM | 🎥 |
| Аудіо | MP3, WAV, OGG | 🎵 |
| Документи | PDF, DOC, DOCX, TXT | 📄 |
| Архіви | ZIP, RAR, 7Z | 📦 |
| Таблиці | XLS, XLSX | 📊 |
| Презентації | PPT, PPTX | 📊 |

## ⚠️ Обмеження

- **Максимальний розмір**: 10 MB на файл
- **Обробка**: Поки що в розробці (файли зберігаються, але не аналізуються)

## 💡 Приклади використання

### Приклад 1: Відправити зображення
```
1. Натисни 📎
2. Вибери картинку
3. Напиши: "Що на цьому зображенні?"
4. Відправ
```

### Приклад 2: Відправити документ
```
1. Натисни 📎
2. Вибери PDF файл
3. Напиши: "Прочитай цей документ"
4. Відправ
```

### Приклад 3: Кілька файлів
```
1. Натисни 📎
2. Вибери кілька файлів (Ctrl + клік)
3. Напиши: "Проаналізуй ці файли"
4. Відправ
```

## 🐛 Проблеми?

### Файл не завантажується
- Перевір розмір (макс 10MB)
- Перевір тип файлу
- Перевір інтернет з'єднання

### Превью не показується
- Оновіть сторінку (Ctrl + Shift + R)
- Перевірте консоль браузера (F12)

### Помилка сервера
- Перевірте чи запущений сервер
- Перевірте логи в консолі

## 📞 Підтримка

Якщо виникли проблеми:
1. Перевір консоль браузера (F12)
2. Перевір логи сервера
3. Перезапусти сервер
4. Очисти кеш браузера
