### ⭐ Базовий мінімум - ВИКОНАНО ✅

**Завдання:**
1. Надіслати перший запит до AI-моделі через API та отримати відповідь
2. Вивести у консоль сирий JSON
3. Витягти з JSON лише текст відповіді і показати його в консоль

**Реалізація:**
- ✅ Метод `send_message()` відправляє запит до GROQ API
- ✅ Виводить повний JSON відповідь з усіма полями
- ✅ Витягує текст з `response.choices[0].message.content`
- ✅ Показує текст окремо в консолі

**Код:**
```python
# Відправка запиту
response = self.client.chat.completions.create(
    model=self.model,
    messages=messages,
    max_tokens=500,
    temperature=0.7
)

# Вивід сирого JSON
response_dict = response.model_dump()
print(json.dumps(response_dict, indent=2, ensure_ascii=False))

# Витяг тексту
assistant_message = response.choices[0].message.content.strip()
print(assistant_message)
```

### ⭐⭐ Середній рівень - ВИКОНАНО ✅

**Завдання:**
1. Додати цикл діалогу, щоб користувач міг надсилати кілька повідомлень підряд
2. Додати команду "вихід", щоб завершити діалог

**Реалізація:**
- ✅ Метод `run()` містить нескінченний цикл `while True`
- ✅ Користувач може писати скільки завгодно повідомлень
- ✅ Команди виходу: `вихід`, `exit`, `quit`, `q`
- ✅ Додаткові команди: `історія`, `очистити`

**Код:**
```python
while True:
    user_input = input("\n👤 Ти: ").strip()
    
    # Команда виходу
    if user_input.lower() in ['вихід', 'exit', 'quit', 'q']:
        print("\n👋 До побачення! Історія збережена.")
        break
    
    # Відправка повідомлення
    response = self.send_message(user_input)
```

### ⭐⭐⭐ Розкішний максимум - ВИКОНАНО ✅

**Завдання:**
1. Додати логування всіх повідомлень у локальний файл `dialog_history.json`
2. Зберігати: дата, роль (user або assistant) і текст повідомлення
3. При наступному запуску асистент може показати попередню історію діалогу

**Реалізація:**
- ✅ Метод `add_to_history()` додає повідомлення в історію
- ✅ Метод `save_history()` зберігає в `dialog_history.json`
- ✅ Метод `load_history()` завантажує при старті
- ✅ Метод `show_history()` показує останні 10 повідомлень
- ✅ Формат: `{"date": "ISO", "role": "user/assistant", "message": "текст"}`

**Структура JSON:**
```json
[
  {
    "date": "2026-02-07T17:30:45.123456",
    "role": "user",
    "message": "Привіт!"
  },
  {
    "date": "2026-02-07T17:30:47.654321",
    "role": "assistant",
    "message": "Привіт! Як справи? 😊"
  }
]
```

**Код:**
```python
def add_to_history(self, role, message):
    entry = {
        "date": datetime.datetime.now().isoformat(),
        "role": role,
        "message": message
    }
    self.dialog_history.append(entry)

def save_history(self):
    with open(self.history_file, 'w', encoding='utf-8') as f:
        json.dump(self.dialog_history, f, ensure_ascii=False, indent=2)

def load_history(self):
    if os.path.exists(self.history_file):
        with open(self.history_file, 'r', encoding='utf-8') as f:
            self.dialog_history = json.load(f)
```
