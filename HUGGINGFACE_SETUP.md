# 🎨 Налаштування Hugging Face для генерації зображень

## ✨ Переваги Hugging Face:
- ✅ **ПОВНІСТЮ БЕЗКОШТОВНО!**
- ✅ Stable Diffusion 2.1 (якість як у DALL-E)
- ✅ Без обмежень на кількість запитів
- ✅ Не потрібна кредитна картка

---

## 🚀 Швидке налаштування (3 хвилини)

### Крок 1: Отримай безкоштовний токен

1. Перейди на https://huggingface.co/
2. Зареєструйся (безкоштовно, email + пароль)
3. Перейди в налаштування: https://huggingface.co/settings/tokens
4. Натисни **"New token"**
5. Назва: `AI Assistant`
6. Тип: **Read** (достатньо для генерації)
7. Скопіюй токен (виглядає як `hf_xxxxxxxxxxxxx`)

### Крок 2: Додай токен в проект

Відкрий файл `.env` і додай:

```env
HUGGINGFACE_API_KEY=hf_твій_токен_тут
USE_IMAGE_GENERATION=True
```

### Крок 3: Перезапусти сервер

```bash
# Зупини поточний сервер (Ctrl+C)
# Запусти знову
python app.py
```

## ⏳ Перший запит

**ВАЖЛИВО:** Перший запит може зайняти 20-30 секунд, бо модель завантажується на сервері Hugging Face. Наступні запити будуть швидші (5-10 секунд)!

Якщо побачиш повідомлення "Модель завантажується", просто зачекай 20 секунд і спробуй ще раз.

## 🔧 Технічні деталі

### Модель
- **Stable Diffusion 2.1** від Stability AI
- Розмір зображень: 512x512 пікселів
- Якість: висока, детальна

### API
- Endpoint: `https://api-inference.huggingface.co`
- Безкоштовно для всіх користувачів
- Без обмежень на запити

---

## ❓ Вирішення проблем

### Помилка: "API токен не налаштовано"
- Перевір що токен доданий в `.env`
- Переконайся що `USE_IMAGE_GENERATION=True`
- Перезапусти сервер

### Помилка: "Модель завантажується"
- Це нормально для першого запиту
- Зачекай 20-30 секунд
- Спробуй ще раз

### Помилка: "Timeout"
- Інтернет з'єднання повільне
- Спробуй ще раз
- Зачекай трохи довше
