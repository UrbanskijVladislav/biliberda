# 🔍 Діагностика проблем з кнопками

## Крок 1: Оновіть браузер

**ВАЖЛИВО!** Використовуйте жорстке оновлення:

- **Windows**: `Ctrl + Shift + R` або `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`
- **Або**: Відкрийте DevTools (F12) → вкладка Network → поставте галочку "Disable cache" → оновіть сторінку

## Крок 2: Відкрийте консоль браузера

1. Натисніть `F12` (або `Ctrl+Shift+I`)
2. Перейдіть на вкладку **Console**
3. Оновіть сторінку

## Крок 3: Перевірте логи

Ви маєте побачити:

```
🔧 Ініціалізація AI Асистента...
📋 Перевірка елементів:
  chatMessages: ✅
  userInput: ✅
  sendBtn: ✅
  loading: ✅
🔗 Прикріплення обробників подій...
✅ Ініціалізація завершена!
```

### Якщо бачите ❌:

Це означає, що елемент не знайдено. Проблема в HTML або ID елементів.

### Якщо не бачите логів взагалі:

JavaScript не завантажується або є помилка синтаксису.

## Крок 4: Перевірте помилки

Шукайте червоні повідомлення в консолі:

### Типові помилки:

**1. "Uncaught TypeError: Cannot read property 'addEventListener' of null"**
- Елемент не знайдено
- Перевірте ID в HTML

**2. "Uncaught SyntaxError"**
- Помилка синтаксису в JavaScript
- Перевірте код на помилки

**3. "Failed to load resource"**
- Файл не завантажується
- Перевірте шлях до файлу

## Крок 5: Тест кнопки

В консолі введіть:

```javascript
// Перевірка об'єкта ui
console.log(ui);

// Перевірка кнопки
console.log(ui.sendBtn);

// Перевірка поля вводу
console.log(ui.userInput);

// Тест відправки
ui.sendMessage();
```

## Крок 6: Тест кліку

1. Введіть текст в поле вводу: "Привіт"
2. Натисніть кнопку відправки
3. В консолі має з'явитися:

```
🖱️ Клік по кнопці відправки
📤 sendMessage викликано
📝 Повідомлення: Привіт
```

## Крок 7: Тест Enter

1. Введіть текст в поле вводу
2. Натисніть Enter
3. В консолі має з'явитися:

```
⌨️ Enter натиснуто
📤 sendMessage викликано
📝 Повідомлення: ...
```

## Можливі проблеми та рішення

### Проблема 1: Кнопка не реагує на клік

**Рішення:**
```javascript
// В консолі
ui.sendBtn.onclick = () => {
    console.log('Клік!');
    ui.sendMessage();
};
```

### Проблема 2: JavaScript не завантажується

**Рішення:**
1. Перевірте мережу (вкладка Network в DevTools)
2. Знайдіть `script.js`
3. Перевірте статус (має бути 200)
4. Якщо 404 - файл не знайдено

### Проблема 3: Кеш браузера

**Рішення:**
1. Очистіть кеш браузера
2. Або відкрийте в режимі інкогніто
3. Або використовуйте жорстке оновлення

### Проблема 4: Конфлікт з іншими скриптами

**Рішення:**
```javascript
// В консолі перевірте
console.log(window.ui); // Має бути об'єкт
```

## Швидка діагностика

Скопіюйте і вставте в консоль:

```javascript
console.log('=== ДІАГНОСТИКА ===');
console.log('ui існує:', typeof ui !== 'undefined');
console.log('sendBtn існує:', ui?.sendBtn !== undefined);
console.log('userInput існує:', ui?.userInput !== undefined);
console.log('sendBtn має onclick:', typeof ui?.sendBtn?.onclick === 'function');

// Тест
if (ui && ui.sendBtn) {
    console.log('✅ Все ОК, тестую відправку...');
    ui.userInput.value = 'Тест';
    ui.sendMessage();
} else {
    console.log('❌ Проблема з ініціалізацією');
}
```

## Якщо нічого не допомагає

1. **Перезапустіть сервер:**
   ```bash
   # Зупиніть (Ctrl+C)
   # Запустіть знову
   python app.py
   ```

2. **Перевірте версію файлів:**
   - Відкрийте `templates/index.html`
   - Знайдіть `script.js?v=32`
   - Має бути версія 32 або вище

3. **Перевірте файл script.js:**
   - Відкрийте `static/script.js`
   - Знайдіть `document.addEventListener('DOMContentLoaded'`
   - Має бути на останніх рядках

4. **Створіть issue:**
   - Скопіюйте всі логи з консолі
   - Скопіюйте помилки (якщо є)
   - Опишіть що робили
