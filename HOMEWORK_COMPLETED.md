## 🧪 Тестування

### Тест 1: Збереження кольору та віку
```
1. "Мій улюблений колір червоний" → Зберігає колір
2. "Мені 16 років" → Зберігає вік
3. "Що ти знаєш про мене?" → Показує колір та вік
4. "Привіт" → Використовує колір у привітанні
```

### Тест 2: Видалення нотаток
```
1. "Запам'ятай що я люблю Python" → Додає нотатку
2. "Запиши що я вчуся в IT Step" → Додає нотатку
3. "Що ти знаєш про мене?" → Показує 2 нотатки
4. "Видали всі нотатки" → Видаляє всі нотатки
5. "Що ти знаєш про мене?" → Показує 0 нотаток
```

### Тест 3: Дати в нотатках
```
1. "Запам'ятай що сьогодні я почав вивчати AI" → Показує [2026-02-07]
2. Перевірити структуру в пам'яті → Має поля date, timestamp, content
```
