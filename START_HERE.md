## 🎯 Як працює автодоповнення?

1. Введіть `/` в поле вводу
2. Побачите список команд
3. Використовуйте ↑↓ для навігації
4. Натисніть Tab для вибору
5. Натисніть Enter для відправки

**Це просто!** 🎊
