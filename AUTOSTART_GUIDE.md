## Метод 1: Task Scheduler (Рекомендовано) ⭐

Найпростіший та найнадійніший спосіб для Windows.

### Крок 1: Запуск скрипта встановлення

1. Клацніть правою кнопкою на файл `install_task_scheduler.bat`
2. Виберіть **"Запустити від імені адміністратора"**
3. Дочекайтеся завершення встановлення

### Крок 2: Перевірка

1. Відкрийте браузер та перейдіть на `http://localhost:5000`
2. Якщо сервер працює - все готово! ✅

### Переваги:
- ✅ Не потребує додаткових програм
- ✅ Автоматичний запуск при вході в систему
- ✅ Легко керувати через Task Scheduler
- ✅ Працює у фоновому режимі

### Керування:

**Відкрити Task Scheduler:**
```cmd
taskschd.msc
```

**Запустити вручну:**
```cmd
schtasks /run /tn "AIAssistant"
```

**Зупинити:**
```cmd
taskkill /f /im pythonw.exe
```

**Видалити:**
```cmd
schtasks /delete /tn "AIAssistant" /f
```

## Метод 2: Windows Service з NSSM

Для більш професійного налаштування.

### Крок 1: Завантаження NSSM

1. Завантажте NSSM з офіційного сайту: https://nssm.cc/download
2. Розпакуйте архів
3. Скопіюйте `nssm.exe` (з папки win64 або win32) в директорію проекту

### Крок 2: Встановлення служби

1. Клацніть правою кнопкою на файл `install_service.bat`
2. Виберіть **"Запустити від імені адміністратора"**
3. Дочекайтеся завершення встановлення

### Переваги:
- ✅ Працює як справжня служба Windows
- ✅ Автоматичний перезапуск при збоях
- ✅ Запускається до входу користувача в систему
- ✅ Професійне керування

### Керування службою:

**Запустити:**
```cmd
net start AIAssistant
```

**Зупинити:**
```cmd
net stop AIAssistant
```

**Перезапустити:**
```cmd
net stop AIAssistant && net start AIAssistant
```

**Видалити:**
```cmd
sc delete AIAssistant
```

**Відкрити менеджер служб:**
```cmd
services.msc
```

## Метод 3: Автозапуск через папку Startup

Найпростіший, але менш надійний спосіб.

### Крок 1: Створення ярлика

1. Клацніть правою кнопкою на файл `start_server.bat`
2. Виберіть **"Створити ярлик"**
3. Скопіюйте створений ярлик

### Крок 2: Додавання в автозапуск

1. Натисніть `Win + R`
2. Введіть: `shell:startup`
3. Натисніть Enter
4. Вставте скопійований ярлик в папку, що відкрилася

### Переваги:
- ✅ Дуже просто налаштувати
- ✅ Не потребує прав адміністратора
- ✅ Легко видалити

### Недоліки:
- ❌ Відкриває вікно консолі
- ❌ Запускається тільки після входу користувача
- ❌ Немає автоматичного перезапуску

## Видалення автозапуску 🗑️

### Автоматичне видалення (всі методи):

1. Клацніть правою кнопкою на файл `uninstall_autostart.bat`
2. Виберіть **"Запустити від імені адміністратора"**
3. Скрипт автоматично видалить всі налаштування

### Ручне видалення:

**Task Scheduler:**
```cmd
schtasks /delete /tn "AIAssistant" /f
```

**Windows Service:**
```cmd
net stop AIAssistant
sc delete AIAssistant
```

**Папка Startup:**
1. Натисніть `Win + R`
2. Введіть: `shell:startup`
3. Видаліть ярлик AI Асистента

## Усунення проблем 🔧

### Проблема: Сервер не запускається

**Рішення:**
1. Перевірте, чи встановлено Python:
   ```cmd
   python --version
   ```

2. Перевірте залежності:
   ```cmd
   pip install -r requirements.txt
   ```

3. Запустіть вручну для перевірки помилок:
   ```cmd
   python app.py
   ```

### Проблема: Порт 5000 зайнятий

**Рішення:**
1. Знайдіть процес, що використовує порт:
   ```cmd
   netstat -ano | findstr :5000
   ```

2. Завершіть процес (замініть PID на номер з попередньої команди):
   ```cmd
   taskkill /PID [номер_процесу] /F
   ```

### Проблема: "Немає прав адміністратора"

**Рішення:**
1. Клацніть правою кнопкою на .bat файл
2. Виберіть **"Запустити від імені адміністратора"**
3. Підтвердіть дію в UAC

### Проблема: Сервер працює, але не відкривається в браузері

**Рішення:**
1. Перевірте, чи працює сервер:
   ```cmd
   netstat -ano | findstr :5000
   ```

2. Спробуйте відкрити:
   - http://localhost:5000
   - http://127.0.0.1:5000

3. Перевірте брандмауер Windows

### Проблема: Сервер зупиняється після закриття консолі

**Рішення:**
Використовуйте Task Scheduler або Windows Service замість ручного запуску.

## Корисні команди 💡

### Перегляд логів Task Scheduler:
1. Відкрийте Event Viewer: `eventvwr.msc`
2. Перейдіть: Windows Logs → Application
3. Фільтруйте за джерелом "Task Scheduler"

### Перегляд логів служби:
1. Відкрийте Event Viewer: `eventvwr.msc`
2. Перейдіть: Windows Logs → System
3. Фільтруйте за джерелом "Service Control Manager"

### Швидкий перезапуск:
```cmd
taskkill /f /im pythonw.exe && schtasks /run /tn "AIAssistant"
```

## Додаткова інформація ℹ️

### Автоматичне відкриття браузера

Якщо хочете, щоб браузер відкривався автоматично, створіть додаткове завдання:

1. Відкрийте Task Scheduler
2. Створіть нове завдання
3. Тригер: При вході в систему (з затримкою 10 секунд)
4. Дія: Запустити програму
   - Програма: `cmd.exe`
   - Аргументи: `/c start http://localhost:5000`

### Зміна порту

Якщо потрібно змінити порт з 5000 на інший:

1. Відкрийте файл `config.py`
2. Знайдіть рядок `PORT = 5000`
3. Змініть на потрібний порт
4. Перезапустіть сервер
