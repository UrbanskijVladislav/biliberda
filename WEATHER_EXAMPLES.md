## 💡 Поради

### 1. Міста з пробілами
Для міст з пробілами просто пиши назву з пробілом:
```
/weather New York
/weather Los Angeles
/weather Mexico City
/weather San Francisco
```

### 2. Різні написання
Якщо місто не знайдено, спробуй:
- Англійську назву замість української
- Офіційну назву замість розмовної
- Додати країну: `/weather Paris France`

### 3. Популярні помилки

❌ **Неправильно:**
```
/weather Нью-Йорк  (кирилиця)
/weather NewYork   (без пробілу)
```

✅ **Правильно:**
```
/weather New York
/weather Kyiv
```

### 4. Без параметра
Якщо не вказати місто, показується погода для Києва:
```
/weather
```

## 🔍 Що робити якщо місто не знайдено?

### Помилка:
```
❌ Місто 'Невідоме' не знайдено. Спробуй інше написання або англійською.
```

### Рішення:
1. **Перевір написання** - можливо є помилка
2. **Спробуй англійською** - `/weather Kyiv` замість `/weather Київ`
3. **Використай офіційну назву** - `/weather Dnipro` замість `/weather Дніпропетровськ`
4. **Додай країну** - `/weather Paris France` якщо є кілька міст з такою назвою

## 📊 Що показує команда?

### 🌡️ Температура
- Реальна температура в градусах Цельсія
- Відчувається як (з урахуванням вітру та вологості)

### ☁️ Опис погоди
Автоматично перекладається українською:
- Sunny → Сонячно
- Cloudy → Хмарно
- Partly cloudy → Частково хмарно
- Rainy → Дощ
- Snowy → Сніг
- Foggy → Туман

### 💧 Вологість
Відсоток вологості повітря (0-100%)

### 💨 Вітер
Швидкість вітру в км/год

### 📍 Локація
Назва міста та країни
