## Можливості асистента

### 💬 Основні функції
- Відповідає на питання про програмування
- Розказує жарти
- Показує поточний час та дату
- Змінює свій настрій
- Веде статистику розмов
- Має швидкі дії для популярних запитів
- **Приймає файли** (зображення, відео, документи, архіви) 📎

### 📎 Робота з файлами
- Підтримка зображень (JPG, PNG, GIF, WebP)
- Підтримка відео (MP4, AVI, MOV, WebM)
- Підтримка документів (PDF, DOC, DOCX, TXT)
- Підтримка архівів (ZIP, RAR, 7Z)
- Максимальний розмір: 10 MB на файл
- Превью прикріплених файлів
- Можливість видалення файлів перед відправкою

## Встановлення та запуск

### 1. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 2. Запуск додатку

```bash
python app.py
```

### 3. Відкрийте браузер

Перейдіть за адресою: `http://localhost:5000`

## Розширення функціональності

Для додавання нових можливостей:

1. **Нові настрої**: Додайте їх у клас `AIAssistant` в `app.py`
2. **Нові відповіді**: Розширте словник `responses` в методі `get_response`
3. **API інтеграція**: Додайте зовнішні API для більш розумних відповідей
4. **База даних**: Інтегруйте SQLite або PostgreSQL для збереження історії
