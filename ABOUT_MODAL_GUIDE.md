# 📖 Інструкція: Модальне вікно "Про нас"

## ✨ Що додано?

Додано красиву кнопку **"Про нас"** (ℹ️) у верхній панелі сайту, яка відкриває стильне модальне вікно з інформацією про AI асистента та розробника.

## 🎨 Особливості

- **Помітна кнопка** з анімацією при наведенні
- **Красиве модальне вікно** з градієнтами та ефектами
- **Плавні анімації** відкриття/закриття
- **Адаптивний дизайн** для всіх пристроїв
- **Легке редагування** біографії через JSON файл

## 📝 Як редагувати біографію?

### Крок 1: Відкрий файл `biography.json`

Файл знаходиться в корені проєкту.

### Крок 2: Відредагуй секцію "developer"

```json
{
  "developer": {
    "title": "Про розробника",
    "content": [
      "Перший параграф твоєї біографії.",
      "Другий параграф з додатковою інформацією.",
      "Третій параграф про твої досягнення.",
      "Можеш додати скільки завгодно параграфів!"
    ]
  }
}
```

### Крок 3: Збережи файл

Після збереження файлу, перезавантаж сторінку в браузері - зміни відобразяться автоматично!

## 📋 Приклад біографії

```json
{
  "assistant": {
```

## 🎯 Що можна налаштувати?

### 1. Біографія асистента
Секція `assistant.content` - масив параграфів про асистента

### 2. Біографія розробника
Секція `developer.content` - масив параграфів про тебе

### 3. Версія та рік
```json
"version": "2.0",
"year": "2024"
```

### 4. Посилання
```json
"links": {
  "github": "твоє посилання на GitHub",
  "documentation": "посилання на документацію",
  "support": "посилання на підтримку або email"
}
```

## 🚀 Як це працює?

1. **Кнопка** в топ-барі (ℹ️) - клік відкриває модальне вікно
2. **JavaScript** завантажує дані з `/biography` API endpoint
3. **Flask** читає файл `biography.json` та повертає дані
4. **Модальне вікно** відображає інформацію з красивим оформленням

## 🎨 Дизайн

- Градієнтні кольори в стилі сайту
- Анімація float для іконки робота
- Плавні переходи та ефекти
- Сітка з можливостями асистента
- Адаптивний дизайн для мобільних

## 🔧 Технічні деталі

### Файли, що були змінені:

1. **templates/index.html** - додано кнопку та модальне вікно
2. **static/style.css** - додано стилі для кнопки та модального вікна
3. **static/script.js** - додано JavaScript для роботи модального вікна
4. **app.py** - додано API endpoint `/biography`
5. **biography.json** - новий файл з біографією

### API Endpoint:

```
GET /biography
```

Повертає JSON з біографією асистента та розробника.
