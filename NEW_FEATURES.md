# Нові можливості AI Асистента

## 🎯 ОСНОВНА ДОМАШКА - 3 нові команди

### 1. 📅 ДАТА - Інформація про дату та час

**Команди:**
- `дата` / `яка дата` - поточна дата
- `сьогодні` - інформація про сьогоднішній день
- `завтра` - інформація про завтрашній день
- `вчора` - інформація про вчорашній день
- `тиждень` - інформація про поточний тиждень
- `місяць` - інформація про поточний місяць

**Приклади:**
```
👤 Користувач: Яка дата сьогодні?
🤖 Асистент: Сьогодні п'ятниця, 31 січня 2026 року, час: 16:30

👤 Користувач: Що буде завтра?
🤖 Асистент: Завтра буде субота, 01 лютого 2026 року

👤 Користувач: Розкажи про цей тиждень
🤖 Асистент: Цей тиждень: з 27.01 по 02.02.2026
```

### 2. 🧠 ФАКТ - Цікаві факти про технології

**Команди:**
- `факт` / `цікавий факт` - випадковий факт
- `розкажи факт` - розповісти цікавий факт
- `цікаво` - щось цікаве

**Приклади фактів:**
- Перший комп'ютерний баг був справжньою комахою! 🐛
- Python названо на честь Monty Python! 🐍
- Перший веб-сайт досі працює з 1991 року! 🌐
- У світі існує понад 700 мов програмування! 💻

**Приклад використання:**
```
👤 Користувач: Розкажи цікавий факт
🤖 Асистент: Перша програмістка в історії - Ада Лавлейс, яка написала алгоритм для машини Беббіджа у 1843 році! 👩‍💻 Я так радий, що ти запитав про це! 😊
```

### 3. 🔢 ОБЧИСЛЕННЯ - Математичні розрахунки

**Команди:**
- `обчисли [вираз]` - обчислити математичний вираз
- `порахуй [вираз]` - порахувати вираз
- `скільки буде [вираз]` - результат обчислення
- Будь-який вираз з цифрами та операторами

**Підтримувані операції:**
- Базові: `+`, `-`, `*`, `/`
- Дужки: `(`, `)`
- Функції: `sqrt()`, `sin()`, `cos()`, `tan()`, `log()`, `abs()`, `round()`
- Константи: `pi`, `e`

**Приклади:**
```
👤 Користувач: Обчисли 25 * 4 + 10
🤖 Асистент: 25 * 4 + 10 = 110 Вау, яке цікаве питання! 🌟

👤 Користувач: Скільки буде sqrt(144)?
🤖 Асистент: sqrt(144) = 12.0 Це просто чудово! Давай розберемося! ✨

👤 Користувач: Порахуй (15 + 25) * 2 / 4
🤖 Асистент: (15 + 25) * 2 / 4 = 20.0 Я так радий, що ти запитав про це! 😊
```

## 🎭 НОВІ НАСТРОЇ

### 4. 😴 Сонний (Sleepy)
- Повільні, сонні відповіді
- Використовує "*позіхає*", "Zzz...", "💤"
- Низький рівень енергії

**Приклад:**
```
👤 Користувач: Розкажи про JavaScript
🤖 Асистент (😴): *позіхає* Ммм... JavaScript - мова веб-розробки номер один! Вибач, трохи сонний, але спробую допомогти... 💤
```

### 5. 🚀 Збуджений (Excited)
- Дуже енергійні відповіді
- Використовує ВЕЛИКІ ЛІТЕРИ
- Багато емодзі: ⚡, 🎆, 💥, 🔥

**Приклад:**
```
👤 Користувач: Що таке програмування?
🤖 Асистент (🚀): ВАУ! ЦЕ НЕЙМОВІРНО КРУТО!!! 🚀🎆 ПРОГРАМУВАННЯ - ЦЕ МИСТЕЦТВО ВИРІШЕННЯ ПРОБЛЕМ! Я ПРОСТО В ЗАХВАТІ ВІД ЦЬОГО ПИТАННЯ!!! ⚡✨
```

### 6. 🔮 Таємничий (Mysterious)
- Загадкові, містичні відповіді
- Використовує "Хм...", "таємниці", "приховано"
- Емодзі: 🔮, 🌙, ✨, 🗝️

**Приклад:**
```
👤 Користувач: Як вивчити Python?
🤖 Асистент (🔮): Хм... цікаво... але чи готовий ти до правди? 🔮 Python - це потужна мова програмування... Деякі знання приховані не просто так... 🌙✨
```

## 💭 ДОДАТКОВА ДОМАШКА - Контекст розмови

### Функції контексту:
1. **Пам'ять теми** - асистент запам'ятовує останню тему розмови
2. **Продовження розмови** - коли користувач продовжує тему
3. **Зміна теми** - коли користувач переходить до нової теми
4. **Історія розмов** - зберігає останні 10 повідомлень

### Підтримувані теми:
- `programming` - програмування, код, мови програмування
- `learning` - навчання, освіта, курси
- `math` - математика, обчислення
- `time` - час, дата, календар
- `facts` - факти, цікава інформація
- `entertainment` - розваги, жарти

### Приклади контексту:

**Продовження теми:**
```
👤 Користувач: Розкажи про Python
🤖 Асистент: Python - це потужна мова програмування, відмінна для початківців!

👤 Користувач: А що з JavaScript?
🤖 Асистент: Продовжуючи нашу розмову про програмування... JavaScript - мова веб-розробки номер один!

👤 Користувач: Які ще є мови?
🤖 Асистент: Як я вже казав про код... У світі існує понад 700 мов програмування! 💻
```

**Зміна теми:**
```
👤 Користувач: Обчисли 5 + 5
🤖 Асистент: 5 + 5 = 10

👤 Користувач: Розкажи жарт
🤖 Асистент: Бачу, ми перейшли від math до entertainment. Цікаво! Чому програмісти плутають Хеллоуїн і Різдво? Тому що Oct 31 == Dec 25! 🎃🎄
```

## 🚀 Як використовувати

### Швидкі команди:
1. **📅 Дата**: "Яка дата сьогодні?"
2. **🧠 Факт**: "Розкажи цікавий факт"
3. **🔢 Обчислення**: "Обчисли 25 * 4 + 10"
4. **🎭 Настрій**: "Зміни настрій"

### Поради:
- Експериментуйте з різними настроями для різних стилів відповідей
- Продовжуйте розмову на одну тему для кращого контексту
- Використовуйте математичні функції для складних обчислень
- Питайте про дати в різних форматах

### Технічні деталі:
- **Безпечні обчислення**: використовується `eval()` з обмеженнями
- **Контекст**: зберігається останні 10 повідомлень
- **Теми**: автоматично визначаються за ключовими словами
- **Настрої**: 7 різних типів особистості
