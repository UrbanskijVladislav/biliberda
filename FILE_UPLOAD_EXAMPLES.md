## 💡 Корисні поради

### 1. Стискай зображення
Використовуй онлайн сервіси для стиснення:
- TinyPNG для PNG
- JPEGmini для JPG
- Squoosh від Google

### 2. Використовуй архіви
Для кількох файлів:
```
Замість: file1.jpg, file2.jpg, file3.jpg (3 файли)
Краще: photos.zip (1 архів)
```

### 3. Перевіряй розмір
Перед завантаженням:
- Windows: Правий клік → Властивості
- Mac: Правий клік → Отримати інформацію
- Linux: ls -lh filename

### 4. Оптимізуй PDF
Використовуй:
- Adobe Acrobat (Зменшити розмір файлу)
- Онлайн сервіси (SmallPDF, iLovePDF)
- Ghostscript (командний рядок)

### 5. Конвертуй відео
Для зменшення розміру:
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec mp3 output.mp4
```

## 🔍 Що далі?

Коли обробка файлів буде готова, ти зможеш:

1. **Аналіз зображень**
   - Розпізнавання об'єктів
   - Опис сцени
   - Пошук схожих зображень

2. **Читання документів**
   - Витяг тексту з PDF
   - Конспектування
   - Пошук інформації

3. **Робота з архівами**
   - Розпакування
   - Перегляд вмісту
   - Пошук файлів

4. **Обробка відео**
   - Витяг кадрів
   - Розпізнавання дій
   - Генерація субтитрів
