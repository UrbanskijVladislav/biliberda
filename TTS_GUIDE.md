# 🔊 Гід по озвучці (Text-to-Speech)

## Що додано?

✅ **AI голос** - природний, приємний голос через ElevenLabs (опціонально)
✅ **Браузерний TTS** - звичайний голос (безкоштовно, завжди працює)
✅ Озвучка відповідей асистента голосом
✅ Підтримка української та англійської мов
✅ Можливість вмикати/вимикати озвучку
✅ Зміна мови через кнопки або текстові команди
✅ **Без збереження файлів** - аудіо передається напряму в браузер

## Два режими озвучки:

### 1. AI голос (ElevenLabs) - РЕКОМЕНДОВАНО 🌟
- Природний, приємний голос
- Звучить як жива людина
- Безкоштовно 10,000 символів/місяць
- Потрібна реєстрація (див. ELEVENLABS_SETUP.md)

### 2. Браузерний TTS - ЗАПАСНИЙ
- Роботизований голос
- Безкоштовно, без обмежень
- Працює без інтернету
- Автоматично вмикається якщо AI недоступний

## Як користуватися?

### Кнопки в інтерфейсі:

1. **🔇/🔊** - Увімкнути/вимкнути озвучку
2. **🇺🇦/🇬🇧** - Змінити мову озвучки

### Текстові команди:

```
"Увімкни озвучку" - увімкнути TTS
"Вимкни озвучку" - вимкнути TTS
"Змінити мову на англійську" - перемкнути на English
"Змінити мову на українську" - перемкнути на українську
```

## Як це працює?

1. Коли асистент відповідає, текст автоматично озвучується (якщо TTS увімкнено)
2. **Спочатку** намагається використати AI голос (ElevenLabs)
3. **Якщо AI недоступний** - автоматично перемикається на браузерний TTS
4. Текст очищається від емодзі перед озвучкою
5. Аудіо передається через base64 (без збереження файлів на диску)

## Налаштування AI голосу:

Див. детальну інструкцію в **ELEVENLABS_SETUP.md**

Коротко:
1. Зареєструйся на https://elevenlabs.io/
2. Отримай API ключ
3. Додай в `.env`:
   ```
   USE_AI_TTS=True
   ELEVENLABS_API_KEY=sk_твій_ключ
   ```
4. Перезапусти сервер

## Підтримка браузерів:

✅ Chrome/Edge - відмінна підтримка
✅ Firefox - хороша підтримка
✅ Safari - базова підтримка
❌ Старі браузери - може не працювати

## Технічні параметри:

**AI голос (ElevenLabs):**
- Модель: eleven_multilingual_v2
- Stability: 0.5
- Similarity: 0.75
- Формат: MP3
- Передача: Base64 (без файлів)

**Браузерний TTS:**
- Швидкість: 1.0 (нормальна)
- Висота: 1.0 (нормальна)
- Гучність: 1.0 (максимальна)

## Приклади використання:

1. Натисни 🔇 щоб увімкнути озвучку
2. Напиши "Привіт" - асистент відповість **природним AI голосом**
3. Натисни 🇬🇧 щоб змінити мову на англійську
4. Напиши "Tell me a joke" - відповідь буде англійською
5. Натисни 🔊 щоб вимкнути озвучку

## Особливості:

- Озвучка зупиняється автоматично при новій відповіді
- Емодзі та спецсимволи видаляються з тексту
- Голос вибирається автоматично залежно від мови
- **Fallback система** - якщо AI не працює, використовується браузерний TTS
- **Без файлів** - аудіо не зберігається на диску, передається напряму

## Що НЕ реалізовано (за завданням):

❌ Голосове введення (voice input) - не потрібно
✅ Озвучка відповідей - реалізовано
✅ AI голос - реалізовано (опціонально)
✅ Зміна мови - реалізовано
✅ Вмикання/вимикання - реалізовано
