# ⚡ ШВИДКИЙ СТАРТ - АНІМАЦІЯ ДРУКУВАННЯ

## 🎯 ЩО ЗРОБЛЕНО:

✅ Додано ефект друкарської машинки для AI відповідей  
✅ Миготливий курсор під час друкування  
✅ Плавне вирівнювання тексту в полі вводу  

## 🚀 ЯК ЗАПУСТИТИ:

### 1. Перезапусти сервер
```bash
python app.py
```

### 2. Жорстке оновлення браузера
**ОБОВ'ЯЗКОВО!**
- Windows/Linux: `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

### 3. Тест
Напиши будь-яке повідомлення і дивись як AI "друкує" відповідь! ⌨️

## 🎬 ЩО ПОБАЧИШ:

1. **Твоє повідомлення** - з'являється миттєво
2. **AI відповідь** - друкується посимвольно з миготливим курсором `|`
3. **Курсор зникає** після завершення друкування

## ⚙️ НАЛАШТУВАННЯ:

### Швидкість друкування:
В `static/script.js` знайди рядок:
```javascript
typeWriter(text, element, cursor, speed = 30)
```

Зміни `30` на:
- `10` - дуже швидко ⚡
- `30` - нормально (за замовчуванням) ✅
- `50` - повільно 🐌
- `100` - дуже повільно 🐢
