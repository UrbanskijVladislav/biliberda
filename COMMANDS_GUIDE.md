# 📋 Посібник з команд AI Асистента

## 🚀 Швидкі команди

AI Асистент тепер підтримує швидкі команди для миттєвого доступу до різних функцій!

### Як використовувати команди?

Просто введіть команду в чат, починаючи з символу `/`. Наприклад: `/help`

### 💡 Автодоповнення

- Почніть вводити `/` і побачите список доступних команд
- Використовуйте стрілки ↑↓ для навігації
- Натисніть `Tab` або клацніть мишкою для вибору команди
- Натисніть `Enter` для відправки

### 🌐 API інтеграції

#### `/weather [місто]`
Показує поточну погоду для будь-якого міста світу!

**Приклади:**
```
/weather Київ
/weather Вінниця
/weather Lviv
/weather London
/weather New York
/weather Tokyo
/weather Paris
/weather
```
(без параметра показує погоду для Києва)

**Підтримка:**
- ✅ Міста України (українською та англійською)
- ✅ Міста світу (англійською)
- ✅ Автоматичний переклад опису погоди
- ✅ Визначення країни

**Відповідь:**
```
☀️ Погода в Київ, Ukraine:
🌡️ Температура: 3°C (відчувається як 0°C)
☁️ Опис: Сонячно
💧 Вологість: 91%
💨 Вітер: 12 км/год
```

**Поради:**
- Для міст з пробілами: `/weather New York`
- Якщо місто не знайдено, спробуй англійську назву
- Працює для будь-якого міста світу!

#### `/news`
Показує останні новини.

**Приклад:**
```
/news
```

#### `/translate [текст]`
Перекладає текст на іншу мову (українська → англійська або навпаки).

**Приклади:**

**Відповідь:**

```

UAH: 41.25
GBP: 0.79
PLN: 4.15

### 🔢 Утиліти

#### `/calc [вираз]`
Обчислює математичний вираз.

**Приклади:**
```
/calc 2+2
/calc 15*7
/calc (100-25)/5
/calc 3.14*2
```

**Відповідь:**
```
🔢 2+2 = 4
```

**Підтримувані операції:**
- Додавання: `+`
- Віднімання: `-`
- Множення: `*`
- Ділення: `/`
- Дужки: `()`
- Десяткові числа: `.` або `,`

#### `/time`

**Приклад:**

/time

```

## 🎯 Поради та трюки

### 1. Швидкий доступ
Натисніть `/` і почніть вводити назву команди - автодоповнення допоможе!

### 2. Комбінування з особистостями
Спробуйте різні команди з різними особистостями:
```
/mood
/joke
```
Кожна особистість розкаже жарт по-своєму!

### 3. Історія команд
Використовуйте стрілки ↑↓ для навігації по історії введених команд.

### 4. Багатомовність
Всі команди працюють однаково українською та англійською мовами.

### 5. Параметри команд
Деякі команди приймають параметри:
- `/weather Париж` - погода в Парижі
- `/calc 100/5` - обчислення
- `/translate Hello` - переклад

## 🔧 Налаштування API

Для повноцінної роботи деяких функцій потрібні API ключі.

### Додавання API ключів

Створіть або відредагуйте файл `.env` в корені проекту:

```env
# Погода (опціонально, працює і без ключа)
OPENWEATHER_API_KEY=your_key_here

# Новини (опціонально)
NEWS_API_KEY=your_key_here
```

### Де отримати ключі?

- **OpenWeatherMap**: https://openweathermap.org/api
- **NewsAPI**: https://newsapi.org/

**Примітка:** Базові функції (погода, переклад, курси валют) працюють без API ключів через безкоштовні сервіси!

## ❓ Часті питання

### Команда не працює?
- Переконайтеся, що команда починається з `/`
- Перевірте правильність написання
- Введіть `/help` для списку команд

### Як очистити чат?
```
/clear
```

### Як змінити мову?
```
/lang
```
або натисніть кнопку 🇺🇦/🇬🇧 в інтерфейсі

### Як дізнатися погоду в іншому місті?
```
/weather Назва_міста
```

### Калькулятор не працює?
Переконайтеся, що використовуєте тільки цифри та операції: `+ - * / ( )`
